 Excel Bike Analysis Project
 Project Overview
This Excel-based project analyzes bike sales data to uncover trends, customer preferences, and performance metrics. The project demonstrates data manipulation, visualization, and analytical skills using multiple sheets and charts.

Objectives
- Analyze sales trends over time.
- Understand customer preferences and behavior.
- Summarize key performance metrics.
- Generate actionable business insights.

Data Description
The project includes:
- Sales Data Sheet: Contains detailed bike sales records.
- Customer Data Sheet:Shows demographic and preference information.
- Summary Sheet: Consolidates insights and key metrics.

Key Features
- Interactive charts and graphs for trend analysis.
- Pivot tables for summarizing data efficiently.
- Highlighted insights for business decision-making.
- Multiple sheets for organized analysis.

Insights & Findings
- Top-selling bikes and categories.
- Seasonal sales trends and patterns.
- Customer preferences and buying behavior.
- Recommendations based on data analysis.

*Tools Used*
- Microsoft Excel
  - Formulas, Pivot Tables, Charts
  - Conditional Formatting
